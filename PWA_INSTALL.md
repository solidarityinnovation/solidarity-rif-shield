# AFGE RIF Shield - iPhone PWA Install Guide

> **IMPORTANT: Must use Safari** - Chrome/Firefox on iOS cannot install PWAs

## Install on iPhone

1. Open **Safari** on iPhone
2. Go to: `https://afge-rif-shield.onrender.com`
3. Tap the **Share button** (box with arrow ↑ at bottom of screen)
4. Scroll down and tap **"Add to Home Screen"**
5. Confirm name **"RIF Shield"** > tap **Add**
6. ✅ Icon appears on home screen - tap to launch full-screen!

## Benefits When Installed
- Full-screen app (no browser bars)
- Works offline
- AFGE branded blue/gold icon
- All data private on your device

## Android
Chrome > 3-dot menu > "Add to Home screen"

## Desktop (Chrome/Edge)
Install icon in address bar > Click Install

## Troubleshooting
- "Add to Home Screen" missing? Use Safari not Chrome
- Offline not working? Open once online to cache assets
